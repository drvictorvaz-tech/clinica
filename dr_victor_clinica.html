🔑 API Key da Anthropic
Obtenha em console.anthropic.com → API Keys → Create Key.
A chave é salva no seu navegador e não sai do seu dispositivo.
⚙️ Parâmetros da análise
Sonnet é ideal para uso diário em clínica. Opus para casos muito complexos.
📖 Como usar
1. Gere sua API key em console.anthropic.com
2. Cole a chave acima e clique em "Salvar e testar"
3. Vá em "Nova Anamnese" e preencha os dados do paciente
4. Clique em "Analisar com IA" no último passo
5. A análise integrativa completa será gerada em segundos

Custo estimado: ~R$ 0,10–0,30 por análise completa
Funciona em: PC, notebook, tablet (qualquer navegador)
