Para ativar o histórico, siga estes passos (5 minutos):

  1. Acesse supabase.com e crie uma conta gratuita
  2. Crie um novo projeto (escolha uma senha forte)
  3. No menu lateral, vá em SQL Editor e execute o SQL abaixo:
CREATE TABLE analises (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  criado_em TIMESTAMPTZ DEFAULT now(),
  atualizado_em TIMESTAMPTZ DEFAULT now(),
  nome_paciente TEXT NOT NULL,
  dados_paciente JSONB DEFAULT '{}',
  resultado JSONB DEFAULT '{}',
  analise_bruta TEXT DEFAULT '',
  notas TEXT DEFAULT ''
);
  1. Em Project Settings → API, copie a Project URL e a anon public key
  2. No Railway, em Variables do seu serviço, adicione:
    SUPABASE_URL = sua Project URL
    SUPABASE_KEY = sua anon public key
Digite o nome e pressione Enter ou clique em Buscar. Deixe em branco para ver todos.
